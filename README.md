## 🚀 Overview
**Jarvis V80** is an elite AI Pilot system designed for high-frequency decision environments. Built upon the **Zenith Core** framework, it bridges the gap between massive scale data mining and nuanced human reasoning. 

Designed specifically with **AI Alignment** in mind, Jarvis V80 ensures that autonomous actions are not just statistically significant, but logically sound and indistinguishable from expert human intelligence.

## 🧠 Technical Architecture: The "8 Generals"
Unlike traditional single-model systems, Zenith Core utilizes an **Ensemble Command Structure**:
* **8 Specialized Models (Generals):** Each general is trained on specific market regime subsets using bootstrapped data.
* **5,000 Decision Trees:** Deep-forest logic execution via hardware-accelerated **XGBoost**.
* **Consensus Logic:** Execution is only triggered when the ensemble reaches a >97% confidence threshold, neutralizing individual model bias.

## 📊 Performance Benchmarks
* **Data Scale:** Orchestration of **2.95 Million+** synchronized data points.
* **Logic Validation:** Achieved a **97% Human-Grade score** on GPTZero logic-nuance benchmarks.
* **Latency:** Sub-millisecond inference time optimized for real-time MT5 integration.

## 📁 Repository Structure
* `/src`: Core logic, including the Zenith Engine and Jarvis Pilot interface.
* `/data`: Samples of the 2.95M operational dataset.
* `/logs`: Audited execution logs (Monthly Battle Logs).
* `requirements.txt`: Environment dependencies.

## 🛠️ Getting Started
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Configure your MT5 credentials in a `.env` file (not included for security).
4. Run the pilot: `python src/main.py`.
